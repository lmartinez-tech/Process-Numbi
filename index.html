Guía para quien presenta

La promesa debe sonar simple

No estamos creando más trabajo. Estamos ordenando cómo acompañamos al cliente para que nadie improvise ni descubra riesgos tarde.

  • Empieza por el dolor: setups confusos generan ansiedad.
  • Luego muestra el mapa: cada persona sabe cuándo participa.
  • Cierra con el criterio de éxito: handoff claro y NPS 9+.
Objetivo de la guía

¿Qué vamos a hacer?

Vamos a ordenar el setup contable para que el equipo sepa qué revisar, qué preguntar, cuándo escalar y cuándo pasar a Operación sin dejar cabos sueltos.

Lectura ejecutiva El proceso no debe sentirse como una auditoría pesada para el cliente. Debe sentirse como acompañamiento: diagnóstico claro, ruta explicada, avances visibles, riesgos conversados a tiempo y cierre formal antes de iniciar la operación recurrente.
Mapa de participación

¿Quién participa y en qué momento?

La idea es que cada persona entienda cuándo entra, qué debe preparar y qué decisión ayuda a tomar. Así el proceso se siente acompañado, no como una lista interminable.
